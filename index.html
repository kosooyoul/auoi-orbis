Auoi Orbis

This is a front-end project for rendering and viewing 360 degree stitched photos inside an orb on legacy web.

How to use?

Tag: div

Attributes:

  • data-view: view name (="auoi-orbis-view")
  • data-src: image url
  • data-json: image data
  • data-fileinput: allow drop file
